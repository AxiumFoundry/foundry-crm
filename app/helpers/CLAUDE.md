# Helpers

## Available Helpers

| Helper | Purpose |
|---|---|
| `ApplicationHelper` | Base helper (currently empty) |
| `CaseStudiesHelper` | Case study view helpers (currently empty) |
| `HealthChecksHelper` | Health check form helpers (currently empty) |
| `PagesHelper` | Page view helpers (currently empty) |

## Conventions

- One responsibility per method
- Keep helpers small and focused
- No database queries in helpers
- Use helpers for view formatting logic only
